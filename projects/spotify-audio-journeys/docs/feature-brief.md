# Spotify Audio Journeys - Feature Brief

## Background
Spotify India is expanding beyond music/podcasts into **Goal-Based Audio Journeys**. The goal is to position Spotify as a personal growth tool for young professionals (18-30) who want to learn skills like English vocabulary, public speaking, or productivity habits through audio.

## MVP Vision
An MVP to be shipped in 3 weeks, targeting 1,000 pilots in Delhi, Bangalore, and Mumbai.

## Core Features (MVP)
1. **Goal Setting**: Users select a learning path (e.g., "Improve English").
2. **Structured Journeys**: Audio tracks containing a mix of songs and bite-sized podcast segments.
3. **Progress Tracker**: Daily streaks and completion percentages.
4. **Nudges**: Smart notifications and motivational reminders.

## Target Audience
- Young professionals & students (18-30).
- Users looking for structured, "passive-to-active" learning.

## Technical Constraints
- AI-driven build (Lovable/Vibe Engine).
- No-code/AI-first backend (Supabase/Airtable).
- 3-week delivery timeline.

## Success Metrics
- **Engagement**: Journey completion rate (e.g., users finishing 5/7 days).
- **Retention**: Return rate specifically for "Audio Journey" vs. standard music streaming.
- **Conversion**: User interest in "Premium Learning" journeys.
